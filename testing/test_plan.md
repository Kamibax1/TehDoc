# План тестирования ConcurrentService

## Цель:
Проверить корректность работы конкурентного сервиса счетчика в многопоточной среде

## Методы тестирования:
- Модульное тестирование - проверка отдельных методов
- Интеграционное тестирование - проверка взаимодействия компонентов
- Функциональное тестирование - проверка бизнес-логики
- Нагрузочное тестирование - проверка производительности

## Участники тестирования
10 одногруппников
